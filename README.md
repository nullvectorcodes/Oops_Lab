# Oops_Lab

This repository contains various Object-Oriented Programming (OOP) experiments in Java. It serves as a practical lab record for foundational Java concepts, from basic syntax to conditional statements.

## How to Clone

To clone this repository to your local machine, run the following command in your terminal:

```bash
git clone https://github.com/nullvectorcodes/Oops_Lab.git
```

## Experiments and Concepts Covered

### Experiment 1
This experiment focuses on the fundamental concepts of Java programming, including basic syntax, data types, operators, and user input.

*   **Exp1.1.java (Hello World):**
    *   **Topic:** Introduction to Java syntax and program structure.
    *   **Concepts Used:** Main method declaration, standard output (`System.out.println`).
*   **Exp1.2.java (Expressions Demo):**
    *   **Topic:** Arithmetic and logical expressions.
    *   **Concepts Used:** Arithmetic operators (`+`, `/`, `*`), boolean logic (`&&`, `||`), variable assignment, scientific notation, operator precedence.
*   **Exp1.3.java (Temperature Converter):**
    *   **Topic:** User input and basic mathematical calculations.
    *   **Concepts Used:** Standard input using `Scanner` class (`Scanner.nextDouble()`), mathematical formulas, string concatenation for output.

### Experiment 2
This experiment dives into control flow utilizing block statements and conditionals.

*   **Exp2.1.java (Check Equal Four):**
    *   **Topic:** Conditional statements for integer values.
    *   **Concepts Used:** Standard input (`Scanner.nextInt()`), `if-else` branching, equality operators (`==`), logical AND operator (`&&`).
*   **Exp2.2.java (Check Double Range):**
    *   **Topic:** Conditional logic for floating-point numbers.
    *   **Concepts Used:** Standard input (`Scanner.nextDouble()`), relational operators (`>`, `<`), compound boolean conditions, `if-else` branching.
